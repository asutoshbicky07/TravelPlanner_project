About the Project
----------------------
Travel Planner is a simple web application that helps users plan their trips by finding routes between locations. It uses the Geoapify API to fetch travel information and displays the results through an easy to use 
Gradio interface.I built this project to learn how to work with APIs, handle environment variables securely, and deploy Python applications using GitHub and Hugging Face Spaces.

Features
------------
Find routes between different locations
View travel distance and route information
Simple and clean user interface
Fast response using the Geoapify API
Secure API key management using environment variables


Technologies Used
-----------------------
Python
Gradio
Requests
Geoapify API
python-dotenv

Notes
------------
The .env file is not included in this repository to keep the API key secure.
For deployment on Hugging Face Spaces, the API key is stored using Secrets instead of a .env file.
